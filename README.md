# Claude Certified Architect Foundations — Prep

This repo is a lightweight, build-as-you-go study kit for the Claude Architect Foundations exam.

## How to use

- Skim `exam-notes/` when learning a topic.
- Drill `flashcards/` daily (add your own Q/A as you miss questions).
- Use `practice/` to track mistakes and build a personal “anti-pattern” list.
- Keep `claude-code-demo/` as a sandbox for Claude Code workflows, rules, and skills.

## Suggested weekly loop

- Read 1 note.
- Create 5 flashcards.
- Answer 10 questions.
- Write a short post-mortem of mistakes.

## Folder map

- `exam-notes/`: structured notes by domain
- `flashcards/`: high-signal terms, traps, best-answer heuristics
- `practice/`: questions, wrong-answer analysis, daily checklist
- `claude-code-demo/`: a tiny codebase to practice workflows and rules
- `mcp-tool-design/`: tool description patterns, error shapes, examples
- `structured-output/`: schemas, validation, retry flows
- `multi-agent-design/`: coordinator patterns, context passing, provenance
