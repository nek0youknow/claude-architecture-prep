# 03 — Claude Code workflows

## Building blocks

- **Commands**: repeatable mini-playbooks (review, test, release)
- **Rules**: project conventions (style, patterns, constraints)
- **Skills**: reusable “how to do X” procedures for the agent

## Common workflows to practice

- Implement → lint/test → review → fix → ship
- Debug from logs → reproduce → minimal fix → regression test
- Refactor: small diffs, safety rails, incremental commits

## Personal workflow notes

- When stuck: reduce scope, isolate variables, add instrumentation.
- Prefer “one file at a time” diffs for reviewability.
